마지막 전체 실행 결과 (2026-07-20): 총 121개 항목 전부 통과 + 축별 외부 검증 3종 추가.
- canon-test (용신 미적용, 관계축만): 천지덕합 1위 60/60, 쌍충 배제 48/48
- 용신 적용 시: 천지덕합 1위 54/60, top3 59/60 — 유일한 이탈(을사→경신 7위)은 신약 을목에 금 과다 상대를 감점하는 억부 규칙의 의도된 결과
- **johu-canon-test (조후축, 외부 표 대조)**: 엔진의 단순화 조후가 궁통보감 120칸 중 여름 90%·겨울 80%(칸에 포함 기준), 주용신 완전 일치는 여름 73%·겨울 60%. 엔진 출력 자체와 표의 일치율 85%(60칸 중 51). **알려진 한계**: 일간 자체가 이미 화(병·정)인데 겨울생이거나, 일간 자체가 이미 화·수·목인데 계절과 우연히 맞물리는 경우(예: 정일간 여름, 계일간 여름) 단순 "계절→반대오행" 규칙이 깨짐 — 궁통보감은 이때 일간을 살리는 다른 오행(예: 화 일간+겨울 → 목으로 화를 지속 공급)을 지정하는데 지금 엔진은 이 예외를 반영하지 않음. 정식 궁통보감 120칸 룩업 테이블로 교체하면 해소되나 별도 구현 필요 (사용자 논의 후 보류 중).
- **monotonic-test (억부축 내부 논리)**: 기신(토)→용신(수) 치환 1381/1381 통과, 기신(토)→희신(목) 치환 2000/2000 통과, 위반 0건. 관계가 같을 때 용신·희신이 늘면 점수가 절대 내려가지 않는다는 억부 방향성이 무작위 대량 케이스에서 완전히 지켜짐.
